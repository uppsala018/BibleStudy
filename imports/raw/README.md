# Raw Imports

Place source text files here before running the import scripts.

## Expected Files

- `imports/raw/kjv/genesis-1.txt`
- `imports/raw/catholic/john-1.txt`

Each file should use one verse per line in this format:

```text
1 In the beginning God created the heaven and the earth.
2 And the earth was without form, and void...
```

## Commands

```bash
npm run import:kjv
npm run import:catholic
```
